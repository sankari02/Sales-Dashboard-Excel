# Sales Dashboard

## Description
An interactive Microsoft Excel dashboard developed to analyze sales performance and provide business insights using Pivot Tables, Charts, and Slicers.

## Features
- Total Sales Analysis
- Profit Analysis
- Average Sales
- Units Sold Analysis
- Interactive Charts and Slicers
- Dynamic Dashboard

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Dashboard Design
- Data Visualization

## Project Outcome
The dashboard provides insights into sales performance, helping identify trends and support business decision-making.
